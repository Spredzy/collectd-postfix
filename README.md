# collectd-postfix

A collectd postfix plugin template

This is a configuration template to collect Postfix statistics.
It is inspired by https://collectd.org/wiki/index.php/Plugin:Tail/Config#Postfix

# Support

  * Basic postfix logs
  * milter-reject rspamd logs
